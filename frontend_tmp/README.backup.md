# ExplainMyBody Frontend

프론트엔드 애플리케이션 (추후 개발 예정)

## 계획된 기능

### 1. 로그인/회원가입 화면
- 이메일/비밀번호 로그인
- 회원가입 시 OCR 입력 가능

### 2. 메인 화면
- 챗봇 선택
- 인바디 정보 확인
- 목표 설정 기능

### 3. 챗봇 화면
- LLM1: 인바디 정보 기반 현재 상태 분석
- LLM2: 목표 입력 후 주간 계획서 생성

### 4. 인바디 정보 화면
- 최신 인바디 정보 표시
- 과거 인바디 정보 목록
- 새 인바디 정보 추가 (수동 입력 또는 OCR)
- 각 기록별 체형 정보 표시
- 인바디 결과 분석 요청 기능

## 기술 스택 (예정)

- React / Vue / Next.js (선택 예정)
- TypeScript
- Tailwind CSS
- Axios (백엔드 API 통신)

## 백엔드 API 연결

백엔드 서버: `http://localhost:8000`
API 문서: `http://localhost:8000/docs`
